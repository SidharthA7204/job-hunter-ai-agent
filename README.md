# 🚀 Job Hunter AI Agent

An automated Python-based Job Search Agent that:
- Searches jobs using Google Jobs (SerpAPI)
- Filters them based on your preferences
- Generates a professional PDF report
- Emails the report directly to you

---

## 📌 Features

✅ Smart job search using SerpAPI  
✅ Custom job preference filtering (skills, salary, location, job type)  
✅ Match scoring system  
✅ Beautiful PDF report generation (ReportLab)  
✅ Automatic email delivery with attachment  
✅ Configurable preferences via JSON file  

---

## 🛠️ Tech Stack

- Python 3.11+
- SerpAPI
- ReportLab (PDF generation)
- python-dotenv
- SMTP (Gmail)
- JSON configuration

## 📊 How It Works

1. Loads job preferences
2. Builds search query
3. Fetches jobs using SerpAPI
4. Filters based on skills, salary, location
5. Generates PDF report
6. Emails report to you
